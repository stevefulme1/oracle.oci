# Code of Conduct

The `oracle.oci` Ansible collection follows the
[Ansible Community Code of Conduct](https://docs.ansible.com/ansible/latest/community/code_of_conduct.html).

## Summary

- Be respectful and considerate.
- Be patient and welcoming to newcomers.
- Be collaborative and constructive.
- Focus on what is best for the community.
- Show empathy towards other community members.

## Reporting

If you experience or witness unacceptable behavior, please report it by
opening a GitHub issue or contacting the maintainers directly.

## Enforcement

Instances of abusive, harassing, or otherwise unacceptable behavior may
result in temporary or permanent bans from the project at the discretion
of the maintainers.
